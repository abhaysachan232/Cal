# React TypeScript Calendar

A simple and reusable Calendar component built with **React + TypeScript**.  
It accepts a `date` prop, renders the correct monthly calendar, highlights the selected date, and supports previous/next month navigation.  
Calendar layout and styling match the provided reference design using plain CSS.

## Features
- Reusable `<Calendar date={...} />` component  
- Correct month/year rendering  
- Weekday headers + aligned dates  
- Active date highlighting  
- Prev/Next month navigation  
- Plain CSS styling  
- Component tests using react-testing-library
